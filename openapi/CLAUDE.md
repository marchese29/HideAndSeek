# OpenAPI Spec

Auto-generated OpenAPI specification from the FastAPI server.

## How It Works

- `openapi/openapi.yaml` is generated by `server/scripts/generate_openapi.py`.
- The pre-commit hook regenerates it automatically when `server/` files change.
- **Do not edit `openapi.yaml` directly** â€” modify the FastAPI routes instead.

## Manual Regeneration

```bash
cd server && uv run python scripts/generate_openapi.py
```
